Featured Projects

Synthetic Animal Motion Generation

Built an end-to-end pipeline to reconstruct 3D quadruped motion from video using segmentation, skeletonization, and SMAL model fitting. The system converts raw video frames into structured joint trajectories that can be used for simulation and analysis. Integrated the reconstructed motion into a Unity-based animation system to generate realistic synthetic data for radar training.

Radiation Signal Classification

Developed a real-time machine learning system for classifying gamma-ray waveforms using a convolutional neural network with uncertainty calibration. Engineered temporal features and optimized inference to achieve sub-millisecond latency for streaming data. Integrated the model with Unity and HoloLens to visualize radiation signals in an interactive 3D environment.
