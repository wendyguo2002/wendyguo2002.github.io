Featured Projects

Synthetic Animal Motion Generation

Built an end-to-end pipeline to reconstruct 3D quadruped motion from video using segmentation, skeletonization, and SMAL model fitting. The system converts raw video into structured joint trajectories for simulation and analysis. Integrated the motion into Unity to generate realistic synthetic data for radar training.

Radiation Signal Classification

Developed a real-time machine learning system for classifying gamma-ray waveforms using a CNN with uncertainty calibration. Optimized for low-latency streaming and improved signal detection accuracy. Integrated with Unity and HoloLens to visualize radiation data in an interactive 3D environment.
