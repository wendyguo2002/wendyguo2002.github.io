Synthetic Animal Motion Generation

Developed an end-to-end pipeline to reconstruct realistic 3D quadruped motion from monocular video, addressing the limited availability of high-quality motion datasets for radar-based learning systems. The pipeline converts raw video frames into structured joint trajectories by combining segmentation, skeletonization, and SMAL model fitting. The reconstructed motion is then integrated into a Unity-based animation system to generate controllable and scalable synthetic motion data.

The system bridges computer vision and simulation by enabling a full workflow from perception to animation, allowing generated motion sequences to be reused for training and evaluation. This approach improves data diversity and provides a practical solution for generating synthetic datasets in scenarios where real-world collection is expensive or infeasible.

Tech: Python, PyTorch, OpenCV, Unity, SMAL

Radiation Signal Classification

Built a real-time machine learning system for classifying gamma-ray waveforms using a convolutional neural network with uncertainty calibration. The system improves signal discrimination in noisy environments and supports continuous low-latency inference for streaming data. By combining learned features with probabilistic calibration, the model provides more reliable predictions than traditional threshold-based methods.

The pipeline was optimized for performance and integrated with Unity and HoloLens to enable interactive 3D visualization of radiation signals. This allows users to observe detected radiation patterns in an augmented reality environment, improving interpretability and real-world usability of the system.

Tech: PyTorch, Unity, HoloLens, C#

AI Portfolio Knowledge Base

Designed a retrieval-augmented generation (RAG) system that transforms static professional documents into a queryable knowledge base. The system enables users to ask complex technical and behavioral questions and receive structured, context-aware responses by combining semantic retrieval with large language model generation.

Implemented a semantic ingestion pipeline with task-aware chunking and embedding-based retrieval using ChromaDB. The system improves response relevance and consistency while maintaining low latency, making it useful for interview preparation and structured knowledge access.

Tech: LlamaIndex, ChromaDB, Transformers

PrairieLearn Feedback Engine

Developed an automated feedback system for proof-block ordering problems within a large-scale educational platform. The system analyzes the logical structure of student submissions and provides deterministic, structured feedback to help users understand correctness and reasoning.

Integrated the feedback engine into PrairieLearn with a Python backend and frontend components, enabling real-time response generation for 800+ students. The system improves scalability of grading workflows while enhancing learning outcomes by providing immediate, structured guidance.

Tech: Python, Mustache.js
