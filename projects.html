Synthetic Animal Motion Generation

Built an end-to-end pipeline to reconstruct 3D quadruped motion from video using segmentation, skeletonization, and SMAL model fitting. The system transforms raw video frames into structured joint trajectories suitable for simulation and analysis. Integrated the reconstructed motion into a Unity-based animation system to generate realistic synthetic motion data for radar training.

Tech: Python, PyTorch, OpenCV, Unity, SMAL

Approach

Video frames are first processed through segmentation to extract animal silhouettes, followed by skeletonization to recover structural shape. 2D keypoints are then detected and used to fit a SMAL model, producing temporally consistent 3D pose sequences. The resulting motion is exported as joint trajectories and used in Unity for motion matching and animation synthesis.

Result

Successfully generated realistic quadruped motion sequences and enabled controllable animation within a simulation pipeline. This provides a scalable way to produce synthetic motion datasets for radar-based learning systems.

Radiation Signal Classification

Developed a real-time machine learning system for classifying gamma-ray waveforms using a convolutional neural network with uncertainty calibration. The system improves signal discrimination and supports continuous streaming inference with low latency. Integrated with Unity and HoloLens to visualize radiation signals in an interactive 3D environment.

Tech: PyTorch, Unity, HoloLens, C#

Approach

Extracted temporal features from waveform signals and trained a CNN to classify signal vs noise. Applied Bayesian uncertainty calibration to improve robustness in ambiguous cases. Optimized the model architecture and inference pipeline to meet real-time constraints.

Result

Achieved high-precision signal classification with significantly improved signal-to-noise ratio. Reduced detection latency and enabled interactive visualization of radiation hotspots in AR.
