Synthetic Animal Motion Generation

Developed an end-to-end pipeline to reconstruct realistic 3D quadruped motion from monocular video, addressing the limited availability of high-quality motion datasets for radar-based learning systems. The system converts raw video frames into structured joint trajectories by combining segmentation, skeletonization, and SMAL model fitting, enabling consistent recovery of pose and motion over time.

The pipeline begins by decomposing videos into frames and extracting animal silhouettes through segmentation. Skeletonization is then applied to capture the structural representation of the animal, followed by 2D keypoint detection to estimate joint locations. These keypoints are used to fit a parametric SMAL model, producing temporally coherent 3D pose sequences suitable for animation and simulation.

To bridge perception and simulation, the reconstructed motion is exported as joint trajectories and integrated into a Unity-based motion matching system. This enables controllable animation synthesis, allowing generated motion to be replayed, extended, and reused in downstream applications such as radar simulation and training data generation.

This approach provides a scalable solution for synthetic motion generation, improving both diversity and controllability compared to real-world datasets. It demonstrates how computer vision pipelines can be tightly coupled with simulation systems to produce high-quality data for machine learning applications.

Tech: Python, PyTorch, OpenCV, Unity, SMAL

Radiation Signal Classification

Built a real-time machine learning system for classifying gamma-ray waveforms using a convolutional neural network enhanced with uncertainty calibration. The system improves signal discrimination in noisy environments and enables more reliable predictions under ambiguous conditions, outperforming traditional threshold-based methods.

The model processes temporal waveform signals and learns discriminative patterns that distinguish valid radiation events from noise. In addition to classification, Bayesian uncertainty calibration is incorporated to estimate prediction confidence, improving robustness and reducing false positives in challenging scenarios.

To support real-time deployment, the model architecture and inference pipeline were optimized for low latency, enabling continuous processing of streaming data. The system achieves near real-time performance, making it suitable for applications requiring rapid and reliable signal interpretation.

The ML pipeline was integrated with Unity and deployed to HoloLens, enabling interactive 3D visualization of radiation signals. Users can observe radiation hotspots and signal patterns in an augmented reality environment, improving interpretability and usability in real-world operational settings.

Tech: PyTorch, Unity, HoloLens, C#

AI Portfolio Knowledge Base

Designed a retrieval-augmented generation (RAG) system that transforms static professional documents into a structured and queryable knowledge base. The system allows users to ask complex technical and behavioral questions and receive context-aware responses by combining semantic retrieval with large language model generation.

A semantic ingestion pipeline was developed to process documents into structured chunks while preserving contextual relationships. Task-aware chunking strategies were applied to maintain coherence across retrieval boundaries, improving the quality and relevance of retrieved information.

The system uses embedding-based retrieval with ChromaDB to identify relevant content, followed by prompt-engineered generation to synthesize structured responses. Additional constraints were introduced to enforce consistent output formats and reduce hallucination, improving reliability for real-world use.

This approach enables efficient knowledge access and significantly improves response relevance compared to keyword-based search. It is particularly useful for interview preparation and structured information retrieval, demonstrating practical applications of LLM-based systems.

Tech: LlamaIndex, ChromaDB, Transformers

PrairieLearn Feedback Engine

Developed an automated feedback system for proof-block ordering problems within a large-scale educational platform. The system analyzes the logical structure of student submissions and provides structured feedback to help users understand correctness and reasoning.

The backend system evaluates the ordering of proof blocks and assigns semantic tags representing logical relationships and correctness. This enables deterministic feedback generation that goes beyond simple correctness checks, offering meaningful guidance to students.

The system was integrated into PrairieLearn using a Python backend and a frontend interface that delivers real-time feedback during problem-solving. The design emphasizes modularity and extensibility, allowing future integration of more advanced reasoning systems.

Deployed to over 800 students, the system significantly reduced manual grading workload while improving learning outcomes. It demonstrates how automated feedback systems can enhance scalability and effectiveness in education.

Tech: Python, Mustache.js
